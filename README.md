# React Theme Switch App

### This project aims at:

- Creating a theme switch app

  - 3 buttons that can toggle theme to light,black,blue
  - Adding env variables to fetch products based on the limit in the env files as per the env

## Getting Started

Clone the repo

```
https://github.com/vimala-siravi/theme-switch.git
```

Install the dependencies

```
npm install
```

Run the project

```
cd theme-switch
npm run start
```
